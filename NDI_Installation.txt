1- Install NDI_ToolBox
2- in Device Manager:
	1- Under Other Devices, right click NDI Host USB Converter or NDI Aurora SCU.
	   Select Update Driver Software... then select Browse my computer for driver software.
	   Select Program Files\Northern Digital Inc\ToolBox\USB Driver. Select Next and install.
	2- After the first set of drivers are installed, browse to Other Devices and right click USB Serial Port.
	   Select Update Driver Software... then select Browse my computer for driver software.
	   Select Program Files\Northern Digital Inc\ToolBox\USB Driver. Select Next and install.
3- Check the Track application

4- Attach the "Ref" to the 1st port and the "Tool" to the 2nd port
